## Структура проекта

Проект состоит из заголовочных файлов и файлов с исходным кодом, включая [`main.cpp`](src/main.cpp), содержащий точку входа программы.

| Директория             | Назначение                                              |
| ---------------------- | ------------------------------------------------------- |
| [`include/`](include/) | заголовочные файлы проекта (API или интерфейс проекта)  |
| [`src/`](src/)         | исходные файлы проекта                                  |

Вам необходимо настроить [QMake](https://doc.qt.io/qt-5/qmake-manual.html) [проект](fatmanager.pro).

*Примечание*. Файл [ресурсов](https://doc.qt.io/qt-5/resources.html) следует размещать в *корне* репозитория.
